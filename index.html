Translation Game for Duolingo

I conceptualized a new feature for a popular product called Duolingo. Here are details of the product and my feature proposal.



    Introduction to Duolingo
    Duolingo is a leading free language-learning platform that offers language courses across 23 different languages. It caters to over 100 million users and is available on desktop, iOS, android and windows platforms. The platform is ad-free and utilizes a crowdsourced business model where users participate in translation and voting on translations. Organizations pay Duolingo to have content translated.

    Opportunities for Improvement
    I have tried using Duolingo and believe it’s a great platform to learn a new language. I love Duo, the owl mascot that asks me to take up lessons everyday. I love the fact that I can add friends and challenge them to compete with me for points. However, in spite of all these hooks to come back to the platform everyday, I find myself slipping away because I don’t prioritize language learning over other activities on a day to day basis. This is because I am not compelled to learn a language as a part of course requirements in school. I am doing it for my own sake, and I lack the motivation to set aside time everyday to learn. However, I still seem to find the time to play games on my mobile phone and browse social media on a daily basis. I believe Duolingo can leverage the human attitude of postponing tasks vs engagement with real-time interactions to create an engaging translation game experience for users.

    Feature: Play Translation Quiz Game with Friends & Other Duolingo Members
    I believe Duolingo can use gamification to make me engage with the platform every single day. It would be great if I could play a translation game real-time with my friends. Even if my friends are not available, I could play the game with any other user on the platform. This feature is different from the feature of challenging my friends to reach a goal because it’s much more real time - I am in the moment, competing with my friends. Also, I often set goals like New Year Resolutions that I don’t follow very meticulously. This feature however, is not about setting a goal and working on it later. It’s about practising more right now.

    Objectives
  • Improve user engagement: To ensure that users spend more time translating words/sentences on Duolingo.
  • Improve user delight: To ensure that learning is made more fun through a game-like experience.
  • Improve speed/quantity of translation: To ensure that users spend more time on Duolingo paying quizzes and in turn, translating sentences. This will result in quicker translations of articles.
  • Attract new users to the platform: This feature is to serve as a bait to attract users who enjoy playing games in general, and are not necessarily on the platform to learn a new language. This may include experts in a language who want to just have some fun.


    • Storyboard: Use case : Sarah challenges her friend Tony to a medium level French quiz on Duolingo.


    Feature Set
    This feature enables users on Duolingo to play a translation game with friends and other users of the platform. The following are the requirements of the feature:
  • A user can initiate a quiz in any language.
  • User can choose the level of the quiz. There are three levels for each quiz - easy, medium and hard.
  • User can choose to play the quiz with a known friend or any other user.
    • -If user chooses to play with a known friend (User selects specific friend for challenge, Friend gets a notification about the request, Friend can accept (now or later) or decline challenge)
    -If user chooses to play with any random user (User is matched to any user who has initiated quiz with same language and level)
    - If no one is available to take up challenge, user is shown a message saying “We’ll notify you as soon as we have matched a player for you.” (In the case of the native mobile app, push notifications are sent to the user, In the case of the desktop version of the app, popup notifications are sent to the user) - Each quiz has ten questions. (Questions are multiple choice questions. This is to remove discrepancy of differences in speed of typing errors while typing on different devices [phone may be tougher to type on than desktop]. Initially, questions will be curated by language specialists at Duolingo. Eventually, questions can be picked from translated article database. Difficulty level of questions can be determined by translation statistics for articles.)
  • There are five quizzes under each difficulty level for each language to begin with. More quizzes will be added eventually.
  • A user can take each quiz only once, to ensure fairness of game.
  • The structure and evaluation of each quiz is as follows:
    • - Each question has a stipulated time in seconds. Time can vary from question to question.
    - Every user is scored based on how many questions are answered correctly and the time taken to answer the question. Player who answers faster gets more points.
    - One user cannot proceed to the next question until the other user has also completed the question or the time limit for that question has been exceeded.
    - User can see his score as well as opponent’s score at each stage of the quiz.
    - Either user can exit the quiz at any time. If a user ends the quiz before completing it, the opponent is the default winner.
    - After game is over, user can challenge same user or some other user to another quiz.
    - Based on the number of quizzes a user wins, user receives virtual trophies.
